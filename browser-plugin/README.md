# MindVault 浏览器插件

选中网页上的任意文本 → 一键保存到 MindVault。

## 安装

1. 打开 Chrome/Edge，进入扩展管理 `chrome://extensions`
2. 打开右上角"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `browser-plugin` 目录

## 使用

1. 在任意网页上选中一段文字
2. 右键 → "保存到 MindVault"（或按 Ctrl+Shift+V）
3. 在弹出的浮窗中编辑内容、添加标签
4. 点"保存"

## 配置

右键点击插件图标 → "选项"，设置 API 地址（默认 `http://localhost:8001/api/v1`）。
