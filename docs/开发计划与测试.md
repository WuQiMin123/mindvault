## 风险与缓解

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|----------|
| 后台任务因服务重启丢失 | 中 | 高 | TaskRunner 启动时恢复未完成任务 |
| SQLite 写入冲突 | 中 | 中 | WAL 模式 + `check_same_thread=False` |
| LLM API 不可用/超时 | 中 | 高 | 指数退避重试 + 明确的错误提示 + 手动重试按钮 |
| B站/知乎 反爬升级 | 低 | 中 | 通用 Fallback 兜底 + 浏览器插件替代方案 |
| 中文内容编码问题 | 低 | 中 | 统一 UTF-8 处理，测试覆盖 CJK 字符 |
| ChromaDB 数据损坏 | 低 | 中 | 向量不备份，可从 content 重建 |

### 步骤 2.2 — B站 Crawler

#### 文件

`backend/app/services/crawler/bilibili.py`

#### 技术路线

```
BV号 → /x/web-interface/view?bvid= → 获取 cid + 标题
   ↓
/x/player/v2?bvid=&cid= → 获取 subtitle 列表
   ↓
取第一个 subtitle URL → 下载 JSON 字幕 → 拼接为纯文本
```

#### 关键代码

```python
class BilibiliCrawler(BaseCrawler):
    async def fetch(self, url: str) -> RawContent:
        bvid = self._extract_bvid(url)
        # 1. 取视频信息
        info = await self._get(f"https://api.bilibili.com/x/web-interface/view?bvid={bvid}")
        cid = info["data"]["cid"]
        title = info["data"]["title"]
        # 2. 找字幕
        subtitles = await self._get(f"https://api.bilibili.com/x/player/v2?bvid={bvid}&cid={cid}")
        if not subtitles["data"]["subtitle"]["subtitles"]:
            raise CrawlerError("该视频没有字幕")
        sub_url = subtitles["data"]["subtitle"]["subtitles"][0]["subtitle_url"]
        # 3. 下载字幕
        sub_data = await self._get(f"https:{sub_url}")
        text = " ".join(item["content"] for item in sub_data["body"])
        return RawContent(title=title, content=text, source="bilibili")
```

```python
def sanitize_content(html: str) -> str:
    """清洗 HTML → 纯文本，防止 XSS"""
    import re
    from html import unescape

    # 1. 移除 script/style 及其内容
    html = re.sub(r'<script[^>]*>.*?</script>', '', html, flags=re.DOTALL)
    html = re.sub(r'<style[^>]*>.*?</style>', '', html, flags=re.DOTALL)
    # 2. 移除所有其他标签
    text = re.sub(r'<[^>]+>', '', html)
    # 3. 解码实体
    text = unescape(text)
    return text.strip()
```

```python
        ...
```

### 步骤 3.3 — 后台任务调度

#### 文件

`backend/app/services/task_runner.py`

#### 职责

- 提交新任务时异步执行完整处理管线
- 服务启动时扫描未完成任务并恢复

#### 关键代码

```python
class TaskRunner:
    def __init__(self, db_session_factory):
        self.db = db_session_factory
        self._tasks: set[asyncio.Task] = set()

    async def start(self):
        """服务启动时，恢复未完成任务"""
        async with self.db() as db:
            result = await db.execute(
                select(Link).where(Link.status.in_(["pending", "processing"]))
            )
            for link in result.scalars():
                link.status = "pending"  # 重置为 pending 重新跑
                await db.commit()
                self._submit(link.id)

    def submit(self, link_id: int):
        """外部接口：提交新任务"""
        self._submit(link_id)

    def _submit(self, link_id: int):
        task = asyncio.create_task(self._run(link_id))
        self._tasks.add(task)
        task.add_done_callback(self._tasks.discard)

    async def _run(self, link_id: int):
        """执行完整管线"""
        async with self.db() as db:
            try:
                link = await db.get(Link, link_id)
                link.status = "processing"
                await db.commit()

                # 1. 抓取
                crawler = CrawlerRegistry.get(link.url)
                result = await crawler.fetch(link.url)
                link.title = result.title
                link.content = sanitize_content(result.content)
                await db.commit()

                # 2. AI 处理
                processor = ContentProcessor(llm_client, db)
                await processor.process(link_id)

            except CrawlerError:
                link.status = "failed"
                link.error = "无法访问该链接"
                await db.commit()
            except LLMError:
                link.status = "failed"
                link.error = "AI 处理失败，请稍后重试"
                await db.commit()
            except Exception as e:
                link.status = "failed"
                link.error = f"未知错误: {str(e)}"
                await db.commit()
```

```python
class VectorStore:
    def __init__(self, path: str):
        self.client = chromadb.PersistentClient(path=path)
        self.collection = self.client.get_or_create_collection("mindvault")

    async def store(self, link_id: int, text: str, embedding: list[float]):
        self.collection.add(
            ids=[str(link_id)],
            embeddings=[embedding],
            metadatas=[{"link_id": link_id}],
            documents=[text],
        )

    async def search(self, embedding: list[float], top_k: int = 5) -> list[SearchResult]:
        results = self.collection.query(query_embeddings=[embedding], n_results=top_k)
        return [SearchResult(...) for ... in results]
```

### 步骤 5.2 — RAG 引擎

#### 文件

`backend/app/services/rag.py`

#### RAG 流程

```python
class RAGEngine:
    async def answer(self, question: str, tag_ids: list[int] | None = None) -> RAGResponse:
        # 1. 问题向量化
        q_embedding = await self.llm.embed(question)
        # 2. 向量检索
        vector_results = await self.vector_store.search(q_embedding)
        # 3. 全文检索 (SQLite FTS)
        fts_results = await self._fulltext_search(question, tag_ids)
        # 4. 合并 + 去重 + 排序
        contexts = self._merge_results(vector_results, fts_results)
        # 5. LLM 综合回答
        answer = await self.llm.chat(question, [c.text for c in contexts])
        return RAGResponse(answer=answer, sources=[c.source for c in contexts])
```
